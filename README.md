# 要件定義
- 背景：前職で毎月の旅費精算が紙媒体での申請だった不便さを感じ、Web申請ができるシステムがあったらと考え「経費精算システム」を制作

- 現状の課題：経費精算を紙で行うため、記入・コピーの時間で休憩時間の5〜10分潰れる
- ゴール：紙を使う手間を省き、Webで申請できる
- アプローチ：
  - 定期券コピー・領収書をとるのが手間
    - 領収書などは写真を撮ってそのままアップロードできる
    - 領収書がPDFの場合もアップロードできる
