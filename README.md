# 経費精算システム
  経費の申請・承認ができるWebアプリケーションです。<br>  
  <img width="1433" alt="トップページ" src="https://user-images.githubusercontent.com/76086661/114688094-58a33400-9d4f-11eb-8211-2a9edd1a41cd.png">

  アプリURL：https://pure-lake-08669.herokuapp.com/ <br>
  ゲストログイン・ゲスト管理者ログインボタンからメールアドレスとパスワードを入力せずにログインできます。

# 要件定義
* 背景：前職で毎月の旅費精算が紙媒体での申請だった不便さを感じ、Web申請ができるシステムがあったらと考え「経費精算システム」を制作

* 現状の課題：経費精算を紙で行うため、記入・コピーの時間で休憩時間の5〜10分潰れる
* ゴール：紙を使う手間を省き、Webで申請できる
* アプローチ：
  - 定期券コピー・領収書をとるのが手間
    - 領収書などは写真を撮ってそのままアップロードできる
    - 領収書がPDFの場合もアップロードできる

### 社員
* 社員がメアドとパスワードでログイン<br>
→月々の消耗品(文房具)や交通費の申請を上げる　／領収書のアップロード<br>
→上司の承認を得る　／承認処理・承認完了メール<br>
* 申請した経費確認　／検索・並べ替え・ページネーション

### 管理者（上司）
* 管理者がメアドとパスワードでログイン<br>
→申請された経費を承認、承認取消（差し戻し）　／承認メール・承認取消メール
* 申請された経費確認　／経費検索・並べ替え・ページネーション
* ユーザ管理　／ユーザ検索・並べ替え・ページネーション
* 一般社員が上司（管理者）になり得る場合　／管理者権限付与
* 経費カテゴリ編集
* 検索画面トグルメニュー
* ユーザ情報編集で管理者からユーザに権限変更できないように　／ラジオボタンでdisabled: trueを条件分岐で追加
