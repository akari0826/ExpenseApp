# 機能一覧
- ユーザ登録、ログイン機能
- 経費データ登録、編集、削除機能
- PDFファイルアップロード機能（Active Storage）
- 管理者機能（管理者権限変更、承認・承認取消）
- 検索機能、並べ替え機能
- ページネーション機能（kaminari）
- 承認メール機能（Action Mailer）
- 論理削除（discard）
